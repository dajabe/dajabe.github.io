What is the difference and why should you care?

Introduction

Good web design requires us to think about how we position elements on the page. You might ask me what an element is and the answer is simply everything. An image on the page, a paragraph of text, a block of color, the menu and the items in it are all examples of elements on a web page. The more information you have on a page the more elements you will have and the more important managing their spacing and presentation becomes. Margin, border, and padding are properties of all HTML elements.

To start us I'll provide you with a simple definition for each of these 3 properties.:

Border
The line that defines the edge of an HTML element
Margin
The spacing between this element and any adjacent elements
Padding
The spacing between the elements border and the conent

Below is a visual representation of these properties. Content in the center is what you are displaying, this could be a paragraph of text, and image, or a link. Around that you have your padding which is the spacing between the border and the content but is also considered part of the content. Then you have your border which allows you to add a border color to help visually seperate or highlight your content. Finally we have the margin which provides spacing an element and other elements.

Margins

For this we will start with margins as I believe that is the easiest property to understand. Take our example boxes below that we will use in our example. We have blue content box and a red content box. Currently they have no padding or margins. Our content is the text stating the color of the box:

Blue Content

Here is the code for the HTML that is used to display this. Simplified a little for this example this will work the same on your own webpage. The div tag used here is the element that we are using in the example but these techniques will work on all HTMl elements. We have given each it's own class so we can apply styles to them seperately.


        <div class="red-box">Red Content</div>
        <div class="blue-box">Blue Content</div>

This is the style's that we have applied to these boxes. We have a section for all div elements and then two sections for specific divs one for our red box and one for our blue box.


        <style>
          div {
            text-align: center;
            font-weight: bold;
          }
          .blue-box {
            background-color: lightblue;
            padding: 0px;
            margin: 0px;
          }
          .red-box {
            background-color: lightcoral;
            padding: 0px;
            margin: 0px;
          }
        </style>

Now we are going to give the red box a margin of 10 pixels. This is a fixed number of pixels on the screen that we want it to be spaced and we are going to apply this margin to the right hand side of the element. 


        <style>
          .red-box {
            background-color: lightcoral;
            padding: 0px;
            margin-right: 10px;
          }
        </style>

Blue Content

So you can see here that we now have a 10px gap between our blue content and our red content. It definitely makes it look a bit better so why don't we apply a 10px margin around all edges for both boxes.


        <style>
          .blue-box {
            background-color: lightblue;
            padding: 0px;
            margin: 10px;
          }
          .red-box {
            background-color: lightcoral;
            padding: 0px;
            margin: 10px;
          }
        </style>
Red Content
Blue Content

This looks great... but wait did you notice something about the gap between the Red and Blue content? It is now bigger and that is because the margin between the two is the sum of both margins. So we now have a 20px gap between our two elements.

We could adjust the margin on the adjacent sides only to bring it closer together or further away but that is out of the scope of this post so we are going to leave it as is.

Borders

Next we are going to tackle borders as it will make it easier for us to see the changes we make with our padding. Borders have a default size of 0px but you can set the size of the border to any width you like. Let's give our border a width of 10px and a solid style for the red and dashed for the blue. We have to specify a color for the border as well and we'll do that with border-color. 


        <style>
          .blue-box {
            background-color: lightblue;
            padding: 0px;
            margin: 10px;
            border: 10px;
            border-style: dashed;
            border-color: yellow;  
          }
          .red-box {
            background-color: lightcoral;
            padding: 0px;
            margin: 10px;
            border: 10px;
            border-style: solid;
            border-color: black;  
          }
        </style>
Red Content
Blue Content

These borders look a little silly but they help show what is happening here. We can see that our margin has not changed but our content has gotten further apart. The new space is filled in with our border. Let's change them to something a bit more appealing and then try another trick with the border. 


        <style>
          .blue-box {
            background-color: lightblue;
            padding: 0px;
            margin: 10px;
            border: 2px;
            border-style: solid;
            border-color: black;
            border-radius: 5px;
          }
          .red-box {
            background-color: lightcoral;
            padding: 0px;
            margin: 10px;
            border: 2px;
            border-style: solid;
            border-color: black;  
            border-radius: 5px;
          }
        </style>
Red Content
Blue Content

Here you can see with used border-radius to add a curve to the corners of our boxes. The content still looks pretty cramped in our boxes so let's move on to padding and do something about that.

Padding

Padding allows us to add spacing around objects to so that the border won't interfer with the content. So let's give our content some room to breathe.


        <style>
          .blue-box {
            background-color: lightblue;
            margin: 10px;
            border: 2px;
            border-style: solid;
            border-color: black;
            border-radius: 5px;

            padding-right: 20px;
          }
          .red-box {
            background-color: lightcoral;
            margin: 10px;
            border: 2px;
            border-style: solid;
            border-color: black;  
            border-radius: 5px;

            padding-bottom: 10px;
          }
        </style>
Red Content
Blue Content

So here we can see that we now have spacing to the right and to the bottom of our two boxes. We probably want padding all the way around the box however so let's change that. Also 10px and 20px both look like a little too much for our boxes so lets drop it back to 5px and see how it looks.


        <style>
          .blue-box {
            background-color: lightblue;
            margin: 10px;
            border: 2px;
            border-style: solid;
            border-color: black;
            border-radius: 5px;

            padding: 5px;
          }
          .red-box {
            background-color: lightcoral;
            margin: 10px;
            border: 2px;
            border-style: solid;
            border-color: black;  
            border-radius: 5px;

            padding: 5px;
          }
        </style>
Red Content
Blue Content

Much better! Now we have a bit of spacing around our content so it is readable and highlighted by our sweet border with it's curved corners. We even have them spaced out from each other so they have some personal space and won't get upset with each other. With some small changes these could easily be a red and blue button that do something fun on your website.
